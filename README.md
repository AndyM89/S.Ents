```bash

# Crée l'environnement virtuel
python -m venv venv

# Active l'environnement
# Sur Windows :
venv\Scripts\activate

# Sur Mac/Linux :
source venv/bin/activate

```

## **les packet a installer**
```bash
pip install --upgrade pip
pip install -r requirements.txt
```
